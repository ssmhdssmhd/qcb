# 超级嗅探 - 视频解析工具

一个基于 PHP 的视频链接嗅探解析工具，支持解析各大视频平台的播放地址。

## 功能特性

- 纯 PHP 实现，无需 Node.js 依赖
- 支持 m3u8 和 mp4 格式视频嗅探
- 双解析方案：cURL 正则解析 + Chrome Headless 嗅探
- 多备用解析接口，提高成功率
- 对外提供统一 API 接口

## 文件结构

```
xt/
├── api.php       # 对外 API 接口（前端调用入口）
├── server.php    # 解析服务端（核心解析逻辑）
└── README.md     # 说明文档
```

## 快速开始

### 环境要求

- PHP 7.4+
- cURL 扩展
- （可选）Chrome/Chromium 浏览器（用于 Headless 嗅探）

### 部署方式

将 `xt` 文件夹上传到 PHP 环境的网站根目录即可。

### 使用方法

**调用接口：**
```
https://你的域名/xt/api.php?url=视频链接
```

**成功返回：**
```json
{
  "code": 200,
  "url": "https://example.com/video.m3u8"
}
```

**失败返回：**
```json
{
  "code": 400,
  "msg": "链接格式不正确"
}
```

## 工作原理

### 方案一：cURL 正则解析（推荐）
1. 使用 cURL 请求第三方解析接口
2. 通过正则表达式从返回的 HTML 中提取视频地址
3. 支持多种匹配模式：video 标签、JS 变量、通用链接、JSON 格式

### 方案二：Chrome Headless 嗅探（备用）
1. 当 cURL 方案失败时自动启用
2. 使用无头浏览器加载解析页面
3. 通过 HAR 日志和 DOM 内容提取视频链接
